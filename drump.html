Tonald Drump 2k16

Overview

Tonald Drump 2k16 is a political satire game remixing elements of classic memory games with politics, developed in 72 hours for Ludum Dare 33 where I served as Project Manager and Lead Programmer. The game was created in Unity with C# and placed 34th out of 1526 games for Humour.
